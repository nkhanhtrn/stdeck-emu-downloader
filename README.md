# ROM Downloader

A Decky Loader plugin for SteamOS that allows searching and downloading ROMs from Myrient.

## Features

- Search and download ROMs from Myrient (https://myrient.erista.me)
- Support for multiple console platforms
- Background download with progress tracking
- Auto-unzip support for compatible formats

## Installation

1. Build the plugin: `npm run build`
2. Package: `./deploy.sh`
3. Install the resulting zip file through Decky Loader

## Development

```bash
# Install dependencies
npm install

# Build for development
npm run build

# Watch for changes
npm run watch
```
